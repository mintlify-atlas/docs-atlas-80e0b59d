---
title: ILogger Interface
description: The core Serilog logging API, used for writing log events.
---

The core Serilog logging API, used for writing log events. The methods on `ILogger` are guaranteed never to throw exceptions.

```csharp
var log = new LoggerConfiguration()
    .WriteTo.Console()
    .CreateLogger();

var thing = "World";
log.Information("Hello, {Thing}!", thing);
```

## Context Methods

Create contextual loggers with additional enrichment.

### ForContext(ILogEventEnricher enricher)

Create a logger that enriches log events via the provided enricher.

<ParamField path="enricher" type="ILogEventEnricher" required>
  Enricher that applies in the context.
</ParamField>

<ResponseField name="returns" type="ILogger">
  A logger that will enrich log events as specified.
</ResponseField>

### ForContext(IEnumerable\<ILogEventEnricher\> enrichers)

Create a logger that enriches log events via the provided enrichers.

<ParamField path="enrichers" type="IEnumerable<ILogEventEnricher>" required>
  Enrichers that apply in the context.
</ParamField>

<ResponseField name="returns" type="ILogger">
  A logger that will enrich log events as specified.
</ResponseField>

### ForContext(string propertyName, object? value, bool destructureObjects = false)

Create a logger that enriches log events with the specified property.

<ParamField path="propertyName" type="string" required>
  The name of the property. Must be non-empty.
</ParamField>

<ParamField path="value" type="object?">
  The property value.
</ParamField>

<ParamField path="destructureObjects" type="bool" default="false">
  If `true`, the value will be serialized as a structured object if possible; if `false`, the object will be recorded as a scalar or simple array.
</ParamField>

<ResponseField name="returns" type="ILogger">
  A logger that will enrich log events as specified.
</ResponseField>

</ResponseField>

## Binding Methods

### BindMessageTemplate(string messageTemplate, object?[]? propertyValues, out MessageTemplate? parsedTemplate, out IEnumerable\<LogEventProperty\>? boundProperties)

Uses configured scalar conversion and destructuring rules to bind a set of properties to a message template. Returns false if the template or values are invalid (ILogger methods never throw exceptions).

<ParamField path="messageTemplate" type="string" required>
  Message template describing an event.
</ParamField>

<ParamField path="propertyValues" type="object?[]?">
  Objects positionally formatted into the message template.
</ParamField>

<ParamField path="parsedTemplate" type="MessageTemplate?" required>
  The internal representation of the template, which may be used to render the boundProperties as text.
</ParamField>

<ParamField path="boundProperties" type="IEnumerable<LogEventProperty>?" required>
  Captured properties from the template and propertyValues.
</ParamField>

<ResponseField name="returns" type="bool">
  True if binding succeeded, false otherwise.
</ResponseField>

```csharp
MessageTemplate template;
IEnumerable<LogEventProperty> properties;
if (Log.BindMessageTemplate("Hello, {Name}!", new[] { "World" }, out template, out properties)
{
    var propsByName = properties.ToDictionary(p => p.Name, p => p.Value);
    Console.WriteLine(template.Render(propsByName, null));
    // -> "Hello, World!"
}
```

### BindProperty(string? propertyName, object? value, bool destructureObjects, out LogEventProperty? property)

Uses configured scalar conversion and destructuring rules to bind a property value to its captured representation.

<ParamField path="propertyName" type="string?" required>
  The name of the property. Must be non-empty.
</ParamField>

<ParamField path="value" type="object?">
  The property value.
</ParamField>

<ParamField path="destructureObjects" type="bool" required>
  If `true`, the value will be serialized as a structured object if possible; if `false`, the object will be recorded as a scalar or simple array.
</ParamField>

<ParamField path="property" type="LogEventProperty?" required>
  The resulting property.
</ParamField>

<ResponseField name="returns" type="bool">
  True if the property could be bound, otherwise false (ILogger methods never throw exceptions).
</ResponseField>
